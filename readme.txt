== Description ==

NINJA Workspace is a revolutionary workspace designed to enhance productivity and keep your clients engaged on your site. With an intuitive interface and powerful features, it streamlines project management and collaboration directly within your WordPress dashboard.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/ninja-workspace` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the Settings->NINJA Workspace screen to configure the plugin.

**From WordPress.org**

1. Download **[NINJA Workspace](https://downloads.wordpress.org/plugin/ninjaworkspace.zip)**
2. Upload the **'ninjaworkspace'** folder to the /wp-content/plugins/ directory

== Frequently Asked Questions ==

= Can I use NINJA Workspace on any WordPress site? =
Yes, NINJA Workspace is designed to work seamlessly with any WordPress site.

= Do I need coding skills to use NINJA Workspace? =
No, NINJA Workspace is user-friendly and requires no coding skills to set up or manage.

== External Services ==

NINJA Workspace relies on the following external service:

### NINJA.net Widget Service

- **Service Description**: This service provides a dynamic workspace widget that enhances user experience by allowing real-time collaboration and project management functionalities.
- **Service URL**: https://worker.nynja.net/v1/Content/NynWidget/nynwidget.js
- **Usage Circumstances**: The service is activated when the user employs the NINJA Workspace widget from their dashboard to access project management and collaboration tools.
- **Terms of Service & Privacy Policy**: Please visit https://ninjaworkspace.com/terms-and-conditions/ and https://ninjaworkspace.com/privacy-policy/ for more information on the terms and privacy practices of the NINJA.net Widget Service.
